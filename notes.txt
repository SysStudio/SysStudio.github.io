_layouts/
    default.html
        Default template
        -contains navbar & such
    news.html
        Inherits layout from default, defines layout of news post page
    page.html
        Inherits layout from default, defines layout of page content
        
_config.yml
    Site configuration and such

Pages don't contain full webpage HTML, but rather are truncated to allow
for ease of editing of the page's content.
    - Full HTML derived from corresponding file in _layouts, corresponding
    MarkDown file, etc.
    - Section not unique to each page
